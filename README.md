# Learn GitHub Actions

This repository contains documentation and examples to help you learn and understand **GitHub Actions**, including workflows, jobs, and common CI/CD use cases.

## Development

### Prerequisites

- Python 3.10+
- `pip`
- Virtual environment (recommended)

### Install Dependencies

Create and activate a virtual environment, then install the required Python dependencies:

```bash
pip install -r requirements.txt
```

### Run Documentation Server Locally

To preview the documentation locally:

```bash
mkdocs serve
```

### Build Documentation (Production)

To generate a production-ready static site:

```bash
mkdocs build
```

The output will be created in the `dist/` directory.
